What to do scroll

Keyboard

Press key Esc - open menu or back to main page
Press key W - hero go forward
Press key S - hero go backward
Press key A - hero go side left
Press key D - hero go side right
Press key Q - hero turn counterclockwise
Press key E - hero turn clockwise

Mouse

Press left button and move - move map
Press right button to map - hero move and attack
Spin the wheel - zoom in or zoom out map
